# Herramientas de Automatización de Redes — Comparación

## Evaluadas

### 1. AgentGPT
- **Tipo**: Agente autónomo de propósito general
- **Pro**: Fácil de usar, genera contenido iterando
- **Contra**: No especializado en redes, necesita integraciones propias
- **Veredicto**: ❌ Demasiado genérico para nuestro caso

### 2. CrewAI
- **Tipo**: Framework multi-agente (Python)
- **Pro**: Equipos de agentes (redactor, estratega, planner), ideal para contenido
- **Contra**: Requiere orquestación propia, no publica directamente
- **Veredicto**: ✅ Interesante para mejorar calidad/estrategia del contenido
- **Acción**: Evaluar como reemplazo del generador de texto en daily-post.py

### 3. n8n
- **Tipo**: Workflow automation (open source, alternativa a Zapier)
- **Pro**: Integraciones nativas con Twitter, IG, LinkedIn, Telegram, Discord
- **Contra**: Visual/drag-and-drop, overhead de infra (servidor dedicado)
- **Veredicto**: ✅ Potente para triggers y respuestas automáticas
- **Acción**: Considerar para respuestas a comentarios/DMs

### 4. Socioboard
- **Tipo**: Social media management (open source)
- **Pro**: Programar posts, multi-cuenta, analytics
- **Contra**: No AI-first, más tradicional, UI pesada
- **Veredicto**: ❌ Postiz ya cubre esto mejor

### 5. Agentfy
- **Tipo**: Multi-agent social media system
- **Pro**: Crawlers de tendencias, analysis, DMs automáticos
- **Contra**: No genera video, APIs directas (complejas), stack pesado
- **Veredicto**: ⚠️ Ideas útiles pero no reemplazo directo
- **Acción**: Extraer patrones de trending analysis
